# MusicWeb - 校园云音乐 (V2.0)

这是一个基于 JavaWeb (JSP+Servlet+JDBC) 开发的音乐管理系统，支持用户上传、管理员审核、在线播放及评论互动。

## ✨ V2.0 新增功能
* **用户系统**：注册、登录、注销。
* **权限管理**：
    * 普通用户：上传音乐、发表评论。
    * 管理员：审核音乐（通过/删除）、管理所有音乐。
* **审核机制**：用户上传的音乐默认为“待审核”状态，需管理员批准后才会在首页显示。
* **评论互动**：登录用户可在播放页发表评论。

## 🛠️ 部署步骤

### 1. 数据库配置
1.  在 MySQL 中运行根目录下的 **`musicdb.sql`** (请确保运行的是最新版脚本)。
2.  脚本会自动创建默认管理员账号。
3.  **修改配置**：打开 `src/main/java/com/music/util/DBUtil.java`，修改 `USER` 和 `PASS` 为你的数据库账号密码。

### 2. Tomcat 配置
1.  添加 Tomcat Local Server (建议版本 9.0)。
2.  Deployment 添加 Artifact: `MusicWeb:war exploded`。
3.  **重要**：Application Context (上下文路径) 必须填 **`/MusicWeb`**。

## 🧪 测试流程 & 默认账号

### 1. 管理员入口 (审核音乐)
* **账号**：`admin`
* **密码**：`123456`
* **操作**：登录后，在首页顶部点击红色的 **[管理后台]**，可以对用户上传的音乐进行“通过”或“删除”操作。

### 2. 普通用户 (上传与评论)
* 请在首页点击“注册”，创建一个新账号进行测试。
* 上传音乐后，需要登录管理员账号去后台点击“通过”，该音乐才会出现在首页榜单中。

## ⚠️ 注意事项
* **JDK 版本**：建议 1.8
* **文件大小**：默认上传限制较大，如遇报错请检查 Tomcat 配置或网络环境。
